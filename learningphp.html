What is PHP?

PHP is a widely-used, open-source scripting language designed for web development. It can be embedded into HTML and is primarily used for server-side web development, creating dynamic pages and interacting with databases.

PHP Full Stack Cheatsheet

Basics

1. Variables and Data Types - Variables store data like strings, integers, and arrays.

<?php $string = "Hello World"; // String $int = 10; // Integer $float = 10.5; // Float $bool = true; // Boolean $array = ["apple", "banana"]; // Array ?>

2. Echo and Print - Output values to the screen.

<?php echo "Hello, World!"; // Outputs text print "PHP is awesome!"; // Another way to output text ?>

3. Conditionals - Make decisions based on conditions.

<?php $age = 20; if ($age >= 18) { echo "You are an adult."; } else { echo "You are a minor."; } ?>

4. Loops - Repeat code multiple times.

<?php for ($i = 0; $i < 5; $i++) { echo $i; } ?>

5. Functions - Group reusable blocks of code.

<?php function greet($name) { return "Hello, $name!"; } echo greet("John"); ?>

6. Arrays - Store multiple values in one variable.

<?php $fruits = ["Apple", "Banana", "Cherry"]; echo $fruits[0]; // Outputs: Apple ?>

7. Array Functions - Functions that manipulate arrays.

<?php $numbers = [1, 2, 3]; array_push($numbers, 4); // Adds 4 to the array print_r($numbers); // Outputs: [1, 2, 3, 4] ?>

8. String Manipulation - Modify strings.

<?php $text = "Hello World"; echo strtoupper($text); // Outputs: HELLO WORLD echo substr($text, 0, 5); // Outputs: Hello ?>

9. Concatenation - Combine strings.

<?php $firstName = "John"; $lastName = "Doe"; echo $firstName . " " . $lastName; // Outputs: John Doe ?>

10. Data Type Conversion - Converting between data types.

<?php $number = "10"; // String $convertedNumber = (int)$number; // Converts to Integer ?>

11. Null Coalescing Operator - Checks if a variable is set and not null.

<?php $name = $user_name ?? 'Guest'; // If $user_name is not set, 'Guest' will be used echo $name; ?>

12. Type Checking - Ensure variables are of a specific type.

<?php $var = 10; if (is_int($var)) { echo "This is an integer."; } ?>

Intermediate

1. Associative Arrays - Use keys and values to store data.

<?php $person = ["name" => "John", "age" => 25]; echo $person["name"]; // Outputs: John ?>

2. Switch Case - A more efficient way to handle multiple conditions.

<?php $day = "Monday"; switch ($day) { case "Monday": echo "Start of the week!"; break; case "Friday": echo "Almost weekend!"; break; default: echo "Midweek."; } ?>

3. Includes - Include files into PHP scripts.

<?php include 'header.php'; // Includes another file require 'footer.php'; // Requires another file ?>

4. Superglobals - Predefined variables in PHP.

<?php $_POST['name'] = "John"; echo $_POST['name']; // Outputs: John ?>

5. Forms Handling - Handle form data submission.

<form method="POST" action="process.php"> Name: <input type="text" name="name"> <input type="submit" value="Submit"> </form>

6. String Functions - Functions that work with strings.

<?php $text = "hello"; echo strlen($text); // Outputs: 5 echo strpos($text, "e"); // Outputs: 1 ?>

7. File Handling - Work with files (read, write, delete).

<?php $file = fopen("example.txt", "w"); fwrite($file, "Hello World!"); fclose($file); ?>

8. Sessions - Store user data across pages.

<?php session_start(); $_SESSION["username"] = "John"; echo $_SESSION["username"]; // Outputs: John ?>

9. Cookies - Store user data on the client side.

<?php setcookie("user", "John", time() + 3600); // Cookie expires in 1 hour echo $_COOKIE["user"]; // Outputs: John ?>

10. Database Connection (MySQLi) - Connecting to a database and performing queries.

<?php $conn = new mysqli("localhost", "root", "", "test_db"); if ($conn->connect_error) { die("Connection failed: " . $conn->connect_error); } $result = $conn->query("SELECT * FROM users"); while ($row = $result->fetch_assoc()) { echo $row["name"]; } ?>

11. Object-Oriented Programming (OOP) - Organizing code using classes and objects.

<?php class Person { public $name; public $age; public function __construct($name, $age) { $this->name = $name; $this->age = $age; } public function greet() { return "Hello, I am $this->name."; } } $person = new Person("John", 25); echo $person->greet(); ?>

12. Namespaces - Prevent naming conflicts in large applications.

<?php namespace MyApp\Models; class User { // Code for User class } ?>

Expert

1. PDO (PHP Data Objects) - Secure database connections.

<?php $pdo = new PDO("mysql:host=localhost;dbname=test", "root", ""); $stmt = $pdo->query("SELECT * FROM users"); while ($row = $stmt->fetch()) { echo $row["name"]; } ?>

2. Prepared Statements - Secure way of executing queries.

<?php $stmt = $pdo->prepare("INSERT INTO users (name, age) VALUES (?, ?)"); $stmt->execute(["John", 25]); ?>

3. Traits - Reusable methods across multiple classes.

<?php trait Logger { public function log($message) { echo $message; } } class User { use Logger; } ?>

4. Exceptions - Handle errors with custom error messages.

<?php try { throw new Exception("Something went wrong!"); } catch (Exception $e) { echo $e->getMessage(); } ?>

5. Custom Exception Handling - Extend built-in exceptions.

<?php class MyException extends Exception {} try { throw new MyException("Custom Exception!"); } catch (MyException $e) { echo $e->getMessage(); } ?>

6. Generators - Create iterators without storing everything in memory.

<?php function myGenerator() { yield 1; yield 2; yield 3; } foreach (myGenerator() as $value) { echo $value; } ?>

7. Magic Methods - Special methods that are automatically called.

<?php class Person { public $name; public function __get($property) { return "Property $property is not defined."; } } $person = new Person(); echo $person->name; ?>

8. Abstract Classes - Define common structure for child classes.

<?php abstract class Animal { abstract public function sound(); } class Dog extends Animal { public function sound() { echo "Bark"; } } $dog = new Dog(); $dog->sound(); ?>

9. Final Keyword - Prevent overriding methods or classes.

<?php final class MyClass {} ?>

10. Dependency Injection - Pass objects to other objects.

<?php class Database { private $pdo; public function __construct(PDO $pdo) { $this->pdo = $pdo; } } $db = new Database($pdo); ?>

11. Reflection - Inspect classes, methods, and properties.

<?php $reflection = new ReflectionClass('MyClass'); echo $reflection->getName(); ?>

12. Performance Optimization - Caching, optimizing database queries, etc.

<?php // Use OPcache for bytecode caching // Cache expensive queries or data to reduce load times ?>
