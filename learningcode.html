HTML Cheatsheet

HTML (HyperText Markup Language) forms the structure of a webpage.

  • Basic Structure:
    • <!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Webpage Title</title>
</head>
<body>
    <h1>Heading</h1>
    <p>This is a paragraph</p>
</body>
</html>
  • Common Elements:
    • Headings: <h1> to <h6>
    • Paragraphs: <p>
    • Lists: Unordered (<ul>) and Ordered (<ol>) Lists
    • Links: <a href="url">Link</a>
    • Images: <img src="image.jpg" alt="Description">
    • Forms: <form> to collect user input (e.g., text, buttons, etc.).
  • HTML5 Features: <video>, <audio>, <canvas>

XML Cheatsheet

XML is used to store and transport data in a structured format.

  • Basic Syntax: Every element must have a start tag and an end tag.
    • <note>
    <to>John</to>
    <from>Jane</from>
    <heading>Reminder</heading>
    <body>Don't forget the meeting tomorrow!</body>
</note>
  • Attributes: Elements can contain attributes like id or class.
  • Namespaces: Define custom elements using namespaces (e.g., xmlns).
  • Uses: XML is used for data storage (e.g., RSS feeds, configuration files).

C++ Cheatsheet

C++ is a powerful, compiled language for system and application software.

  • Basic Structure: #include and main() function.
    • #include <iostream>
using namespace std;
int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
  • Variables and Types: int, char, float, double, string
  • Control Flow: if, for, while, switch.
  • Functions: Defined using return_type function_name(params)
    • int add(int a, int b) {
    return a + b;
}
  • Object-Oriented Programming: Classes, Inheritance, Polymorphism
  • STL (Standard Template Library): Vectors, Maps, Sets

Java Cheatsheet

Java is a class-based, object-oriented programming language used for building cross-platform applications.

  • Basic Structure:
    • public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
  • Variables and Types: int, double, char, String, boolean
  • Control Structures: if, else, switch, for, while
  • Functions: Defined inside classes. No need for return_type function_name(params) in main method.
    • public int add(int a, int b) {
    return a + b;
}
  • OOP Features: Classes, Objects, Inheritance, Polymorphism, Abstraction
  • Advanced Features: Exception Handling (try-catch), Threads, Generics, Lambda Expressions
  • Popular Libraries: JavaFX, Spring Framework, Apache Commons

JavaScript Cheatsheet

JavaScript is a versatile, interpreted language primarily used for creating interactive web pages.

  • Basic Structure:
    • <script>
    console.log("Hello, World!");
</script>
  • Variables: var, let, const
  • Functions: Defined using function keyword, or arrow functions.
    • function greet(name) {
    return "Hello, " + name;
}
                    
    const add = (a, b) => a + b;
  • DOM Manipulation: Select and manipulate elements using document.getElementById(), document.querySelector()
  • Events: Attach event listeners to HTML elements.
    • document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});
  • Async Programming: setTimeout(), setInterval(), Promises, async/await
  • Popular Libraries: React.js, Vue.js, Node.js

Ruby Cheatsheet

Ruby is an interpreted, high-level programming language known for its simplicity and productivity.

  • Basic Syntax:
    • puts "Hello, World!"
  • Variables and Data Types: Integers, Strings, Arrays, Hashes
    • x = 10
name = "John"
array = [1, 2, 3]
hash = { key: "value" }
  • Control Flow: if, else, elsif, unless, case
  • Loops: while, until, for, each
    • array.each do |element|
  puts element
end
  • Functions: Defined with the def keyword
    • def greet(name)
    puts "Hello, #{name}!"
end
  • OOP in Ruby: Classes, Modules, Inheritance, Polymorphism
  • Popular Libraries: Rails, Sinatra

React.js Cheatsheet

React is a JavaScript library for building user interfaces, particularly single-page applications.

  • Basic Setup: React components are functions or classes.
    • import React from "react";

function App() {
    return <div>Hello, React!</div>
}
export default App;
  • JSX Syntax: JSX allows HTML-like syntax inside JavaScript.
    • const element = <h1>Hello, world!</h1>
  • State and Props: State is local data, Props are data passed to components.
    • const [count, setCount] = useState(0);
  • Effect Hook: useEffect for side effects like API calls.
    • useEffect(() => {
    document.title = `You clicked ${count} times`;
}, [count]);
  • Popular Tools: React Router, Redux

Angular Cheatsheet

Angular is a platform and framework for building single-page client applications using HTML and TypeScript.

  • Basic Structure: Angular applications are composed of components and modules.
    • import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  template: '<h1>Hello, Angular!</h1>',
  styleUrls: ['./app.component.css']
})
export class AppComponent {}
  • Directives: Structural directives like *ngIf and *ngFor
    • <div *ngIf="isVisible">This is visible!</div>
<div *ngFor="let item of items">{{ item }}</div>
  • Forms: Template-driven and reactive forms for handling user input.
  • Services and Dependency Injection: Use services to share data and logic across components.
    • @Injectable()
export class DataService {
  getData() {
    return ['Angular', 'React', 'Vue'];
  }
}
  • Routing: Use Angular Router for navigation between components.

PHP Cheatsheet

PHP is a server-side scripting language designed for web development but also used as a general-purpose language.

  • Basic Syntax:
    • <?php
    echo "Hello, World!";
?>
  • Variables and Data Types: String, Integer, Float, Boolean, Array, Object
    • $name = "John";
$age = 30;
$items = array("apple", "banana");
  • Control Flow: if, else, elseif, switch
  • Loops: while, for, foreach
  • Functions: Defined using function keyword
    • function greet($name) {
    return "Hello, $name!";
}
  • Common Functions: strlen(), array(), $_GET, $_POST
  • OOP: Classes, Objects, Inheritance, Interfaces, Traits

Golang Cheatsheet

Go (or Golang) is a statically typed, compiled language designed for simplicity and efficiency, especially in cloud and server-side applications.

  • Basic Syntax:
    • package main
import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
  • Variables: var, const
    • var x int = 10
const y = 5
  • Functions: Function declaration using func
    • func add(a int, b int) int {
    return a + b
}
  • Control Structures: if, for, switch, select
    • if x > y {
    fmt.Println("x is greater than y")
} else {
    fmt.Println("y is greater than x")
}
                    
    for i := 0; i < 5; i++ {
    fmt.Println(i)
}
  • Goroutines and Channels: Used for concurrency. Goroutines are lightweight threads, and channels are used to communicate between them.
    • go func() {
    fmt.Println("Hello from a goroutine")
}()

ch := make(chan string)
go func() {
    ch <- "Hello from channel"
}()
fmt.Println(<-ch)
  • Structs: Define custom data types.
    • type Person struct {
    Name string
    Age  int
}
  • Popular Libraries: Gin, Gorm, Go kit
