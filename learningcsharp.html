C# (pronounced "C-sharp") is a modern, object-oriented programming language developed by Microsoft. It is widely used for building applications ranging from simple desktop programs to complex enterprise-level web and mobile applications. Known for its simplicity and versatility, C# is a popular choice for developing games (via Unity), creating APIs, and building software for Windows.

    Console.WriteLine("Number is greater than 5");

    Console.WriteLine("Number is 5");
else
    Console.WriteLine("Number is less than 5");

{

}

Inheritance and Polymorphism

class Animal
{
    public virtual void Speak()
    {
        Console.WriteLine("Animal speaks.");
    }
}

class Dog : Animal
{
    public override void Speak()
    {
        Console.WriteLine("Dog barks.");
    }
}

Animal myDog = new Dog();
myDog.Speak(); // Output: Dog barks.

Abstract Classes and Interfaces

abstract class Shape
{
    public abstract double GetArea();
}

class Circle : Shape
{
    public double Radius { get; set; }
    public override double GetArea() => Math.PI * Radius * Radius;
}

Circle circle = new Circle { Radius = 5 };
Console.WriteLine(circle.GetArea());

Dependency Injection

public interface IService
{
    void Serve();
}

public class Service : IService
{
    public void Serve() => Console.WriteLine("Service Called");
}

public class Consumer
{
    private readonly IService _service;

    public Consumer(IService service) => _service = service;

    public void Start() => _service.Serve();
}

IService service = new Service();
Consumer consumer = new Consumer(service);
consumer.Start();

Multithreading

using System.Threading;

void PrintNumbers()
{

    {

Singleton Pattern

class Singleton
{
    private static Singleton _instance;

    private Singleton() { }

    public static Singleton Instance => _instance ??= new Singleton();
}

Singleton singleton1 = Singleton.Instance;
Singleton singleton2 = Singleton.Instance;

Console.WriteLine(object.ReferenceEquals(singleton1, singleton2)); // True

Compilers for C#

Below is a list of popular compilers and tools for C# development:

  • Microsoft Visual C# Compiler (csc): The default C# compiler that comes with the .NET SDK. It's used for compiling C# applications into .NET executables or libraries.
  • Mono Compiler: An open-source, cross-platform C# compiler that allows developers to build and run C# applications on multiple platforms, including Linux and macOS.
  • Roslyn (Microsoft Code Analysis): A compiler as a service that provides rich code analysis and is used in Visual Studio and other tools. It includes both the C# and Visual Basic compilers.
  • JetBrains Rider: A cross-platform IDE for .NET development that includes a powerful built-in C# compiler.
  • DotGNU Portable.NET: A discontinued project that provided an alternative implementation of the CLI and a C# compiler.
  • Online C# Compilers: Several online tools, such as DotNetFiddle and Replit, allow you to write and compile C# code without installing a compiler.
