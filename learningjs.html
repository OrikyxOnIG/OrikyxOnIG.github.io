What is JavaScript?

JavaScript is a versatile, high-level programming language primarily used for creating dynamic and interactive content on websites. It allows developers to implement complex features such as interactive forms, animations, and dynamic content updates. JavaScript can be used on the client side and server side, and it integrates seamlessly with HTML and CSS to build responsive, engaging web applications.

JavaScript Learning Guide

Basic

1. Variables - Storing values using var, let, or const.

// Variable declaration var x = 10; let y = 20; const z = 30;

2. Data Types - Common types include strings, numbers, booleans, and objects.

let name = "John"; // String let age = 25; // Number let isAdmin = true; // Boolean let user = { name: "John", age: 25 }; // Object

3. Functions - Reusable blocks of code.

function greet(name) { return "Hello " + name; } console.log(greet("Alice")); // Outputs: Hello Alice

4. Conditional Statements - Decision-making structures.

let age = 18; if (age >= 18) { console.log("You are an adult."); } else { console.log("You are a minor."); }

5. Loops - Repeat code with for, while, or do-while.

for (let i = 0; i < 5; i++) { console.log(i); // Outputs: 0, 1, 2, 3, 4 }

6. Arrays - Store multiple values in one variable.

let fruits = ["apple", "banana", "cherry"]; console.log(fruits[0]); // Outputs: apple

7. Array Methods - Manipulate arrays with push, pop, etc.

let numbers = [1, 2, 3]; numbers.push(4); // Adds 4 to the array console.log(numbers); // Outputs: [1, 2, 3, 4]

8. String Methods - Work with strings using length, toUpperCase, etc.

let text = "hello world"; console.log(text.toUpperCase()); // Outputs: HELLO WORLD console.log(text.length); // Outputs: 11

9. Template Literals - Use backticks for dynamic strings.

let name = "John"; console.log(`Hello, ${name}!`); // Outputs: Hello, John!

10. Basic DOM Manipulation - Modify HTML elements.

document.getElementById("demo").innerHTML = "Hello, JavaScript!";

11. Events - Respond to user interactions.

document.getElementById("btn").addEventListener("click", function() { alert("Button clicked!"); });

12. Comments - Add comments to your code.

// This is a single-line comment /* This is a multi-line comment */

13. Console - Debug code with console.log.

console.log("Debugging info here.");

14. Type Checking - Verify variable types.

console.log(typeof 42); // Outputs: number console.log(typeof "Hello"); // Outputs: string

Beginner

1. Let and Const - Block-scoped variables.

let score = 10; // Mutable const PI = 3.14; // Immutable

2. Functions with Parameters - Pass arguments to functions.

function multiply(a, b) { return a * b; } console.log(multiply(5, 4)); // Outputs: 20

3. Arrow Functions - Concise syntax for functions.

const add = (a, b) => a + b; console.log(add(3, 7)); // Outputs: 10

4. Objects - Store key-value pairs.

let car = { brand: "Toyota", model: "Corolla" }; console.log(car.brand); // Outputs: Toyota

5. Object Destructuring - Extract properties from objects.

const { brand, model } = car; console.log(brand); // Outputs: Toyota

6. Spread Operator - Expand arrays or objects.

let numbers = [1, 2, 3]; let moreNumbers = [...numbers, 4, 5]; console.log(moreNumbers); // Outputs: [1, 2, 3, 4, 5]

7. Template Literals with Loops - Generate dynamic HTML.

let items = ["apple", "banana"]; items.forEach(item => console.log(`Item: ${item}`));

8. Event Listeners - React to DOM events.

document.querySelector("button").addEventListener("click", () => { console.log("Button Clicked"); });

9. Basic Form Validation - Check user input.

const input = document.querySelector("input"); if (input.value === "") { alert("Input cannot be empty"); }

10. Default Parameters - Set defaults for function arguments.

function greet(name = "Guest") { console.log(`Hello, ${name}`); } greet(); // Outputs: Hello, Guest --- ### **Intermediate** ```html

Intermediate

1. Promises - Handle asynchronous tasks.

const promise = new Promise((resolve, reject) => { resolve("Success"); }); promise.then(result => console.log(result));

2. Async/Await - Simplified promise handling.

async function fetchData() { const data = await fetch("https://api.example.com"); console.log(data); } fetchData();

3. Modules - Use export and import.

// File: math.js export const add = (a, b) => a + b; // File: main.js import { add } from './math.js'; console.log(add(2, 3)); // Outputs: 5

4. Classes - Create objects with shared behavior.

class Animal { constructor(name) { this.name = name; } speak() { console.log(`${this.name} makes a sound.`); } } let dog = new Animal("Dog"); dog.speak();

5. Inheritance - Extend classes.

class Dog extends Animal { bark() { console.log("Woof!"); } } let myDog = new Dog("Buddy"); myDog.bark();

6. Error Handling - Use try...catch.

try { let result = x / 0; } catch (error) { console.error("An error occurred:", error.message); }

7. Fetching APIs - Use Fetch API to get data.

fetch("https://api.example.com") .then(response => response.json()) .then(data => console.log(data));

8. Local Storage - Store data in the browser.

localStorage.setItem("name", "John"); console.log(localStorage.getItem("name"));

9. Session Storage - Temporary storage.

sessionStorage.setItem("session", "active"); console.log(sessionStorage.getItem("session"));

10. Timers - Delay tasks with setTimeout and setInterval.

setTimeout(() => console.log("This runs after 2 seconds"), 2000); setInterval(() => console.log("Repeating every 1 second"), 1000);

Advanced

1. Closures - Functions with access to their outer scope.

function outer() { let count = 0; return function inner() { count++; console.log(count); }; } const counter = outer(); counter(); // Outputs: 1 counter(); // Outputs: 2

2. Currying - Transforming functions to take arguments one at a time.

function add(a) { return function(b) { return a + b; }; } console.log(add(5)(3)); // Outputs: 8

3. Higher-Order Functions - Functions that take or return other functions.

function higherOrder(fn) { return function(x) { return fn(x); }; } const double = higherOrder(x => x * 2); console.log(double(4)); // Outputs: 8

4. Debouncing - Limit the rate of a function execution.

function debounce(fn, delay) { let timer; return function(...args) { clearTimeout(timer); timer = setTimeout(() => fn(...args), delay); }; }

5. Throttling - Control function execution at regular intervals.

function throttle(fn, limit) { let inThrottle; return function(...args) { if (!inThrottle) { fn(...args); inThrottle = true; setTimeout(() => (inThrottle = false), limit); } }; }

6. Event Delegation - Attach events to a parent element.

document.querySelector("#parent").addEventListener("click", (e) => { if (e.target && e.target.tagName === "BUTTON") { console.log("Button clicked!"); } });

7. Modules with Dynamic Imports - Import modules on demand.

import("./module.js").then((module) => { module.someFunction(); });

8. WeakMap and WeakSet - Collections for garbage collection.

let weakMap = new WeakMap(); let obj = {}; weakMap.set(obj, "value"); console.log(weakMap.get(obj)); // Outputs: value

9. Symbol - Unique, immutable values.

let sym1 = Symbol("id"); let sym2 = Symbol("id"); console.log(sym1 === sym2); // Outputs: false

10. Prototypes - Use prototypes for inheritance.

function Person(name) { this.name = name; } Person.prototype.greet = function() { console.log(`Hello, ${this.name}`); }; const john = new Person("John"); john.greet(); // Outputs: Hello, John

11. Custom Iterators - Create custom iterable objects.

const myIterable = { *[Symbol.iterator]() { yield 1; yield 2; yield 3; }, }; console.log([...myIterable]); // Outputs: [1, 2, 3]

12. Generator Functions - Create iterable sequences.

function* generator() { yield 1; yield 2; yield 3; } const gen = generator(); console.log(gen.next().value); // Outputs: 1

13. Async Generators - Handle asynchronous iteration.

async function* asyncGenerator() { yield Promise.resolve(1); yield Promise.resolve(2); } (async () => { for await (let value of asyncGenerator()) { console.log(value); // Outputs: 1, 2 } })();

14. Proxy - Intercept object operations.

const handler = { get: (target, prop) => { return prop in target ? target[prop] : "Property does not exist"; }, }; const obj = new Proxy({ name: "John" }, handler); console.log(obj.name); // Outputs: John console.log(obj.age); // Outputs: Property does not exist

Expert

1. Memoization - Optimize functions by caching results.

function memoize(fn) { const cache = {}; return function (...args) { const key = JSON.stringify(args); if (!cache[key]) { cache[key] = fn(...args); } return cache[key]; }; } const add = memoize((a, b) => a + b); console.log(add(2, 3)); // Outputs: 5 console.log(add(2, 3)); // Cached: 5

2. Dynamic Function Creation - Use the Function constructor.

const dynamicFunc = new Function('a', 'b', 'return a + b;'); console.log(dynamicFunc(5, 3)); // Outputs: 8

3. Dynamic Imports - Import modules dynamically.

(async () => { const module = await import('./module.js'); module.someFunction(); })();

4. Module Pattern - Encapsulation using closures.

const Module = (function () { let privateVar = "I am private"; return { getPrivate: function () { return privateVar; }, }; })(); console.log(Module.getPrivate()); // Outputs: I am private

5. Event Loop - Understanding async behavior.

console.log("Start"); setTimeout(() => console.log("Middle"), 0); console.log("End"); // Outputs: Start, End, Middle

6. Memory Management - Avoid memory leaks using weak references.

let weakRef = new WeakMap(); let obj = {}; weakRef.set(obj, "value"); console.log(weakRef.get(obj)); // Outputs: value

7. Advanced Promises - Handle multiple promises with Promise.allSettled.

const promises = [ Promise.resolve(10), Promise.reject("Error"), Promise.resolve(20), ]; Promise.allSettled(promises).then((results) => console.log(results));

8. Advanced Regular Expressions - Use named capture groups.

const regex = /(?\\d{4})-(?\\d{2})-(?\\d{2})/; const match = regex.exec("2024-12-30"); console.log(match.groups); // Outputs: { year: "2024", month: "12", day: "30" }

9. Design Patterns - Implement Singleton Pattern.

const Singleton = (function () { let instance; function createInstance() { return { id: Math.random() }; } return { getInstance: function () { if (!instance) { instance = createInstance(); } return instance; }, }; })(); const instance1 = Singleton.getInstance(); const instance2 = Singleton.getInstance(); console.log(instance1 === instance2); // Outputs: true

10. Advanced Debugging - Use debugger statements effectively.

function debugExample() { debugger; // Execution pauses here let x = 10; return x * 2; } debugExample();

11. Proxy with Reflect API - Intercept and handle object operations.

const handler = { get(target, prop) { return Reflect.get(target, prop) || "Not Found"; }, }; const proxy = new Proxy({ a: 1 }, handler); console.log(proxy.a); // Outputs: 1 console.log(proxy.b); // Outputs: Not Found

12. Advanced Iterables - Customize iterables with the Symbol.iterator.

const advancedIterable = { items: ["a", "b", "c"], [Symbol.iterator]: function* () { for (const item of this.items) { yield item.toUpperCase(); } }, }; console.log([...advancedIterable]); // Outputs: ["A", "B", "C"]

13. Web Workers - Run scripts in background threads.

// worker.js onmessage = function (e) { postMessage(e.data * 2); }; // main.js const worker = new Worker("worker.js"); worker.postMessage(10); worker.onmessage = function (e) { console.log(e.data); // Outputs: 20 };

14. Intl API - Format dates, numbers, and currencies.

const formatter = new Intl.NumberFormat("en-US", { style: "currency", currency: "USD" }); console.log(formatter.format(12345.67)); // Outputs: $12,345.67

Master

1. Custom Event Emitters - Create your own event emitters.

class EventEmitter { constructor() { this.events = {}; } on(event, listener) { if (!this.events[event]) { this.events[event] = []; } this.events[event].push(listener); } emit(event, ...args) { if (this.events[event]) { this.events[event].forEach(listener => listener(...args)); } } } const emitter = new EventEmitter(); emitter.on("data", (msg) => console.log(msg)); emitter.emit("data", "Hello, Event!"); // Outputs: Hello, Event!

2. Advanced Async Patterns - Implement Async/Await with Error Handling.

3. Custom Promises - Implement your own promise library.

class MyPromise { constructor(executor) { this.callbacks = []; this.state = "pending"; this.value = null; const resolve = (value) => { this.state = "fulfilled"; this.value = value; this.callbacks.forEach(cb => cb(value)); }; executor(resolve); } then(callback) { if (this.state === "fulfilled") { callback(this.value); } else { this.callbacks.push(callback); } } } const promise = new MyPromise(resolve => { setTimeout(() => resolve("Done!"), 1000); }); promise.then(console.log); // Outputs: Done!
