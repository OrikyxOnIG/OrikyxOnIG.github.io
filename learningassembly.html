Learning Assembly

Home

Assembly is a low-level programming language that is designed for a specific computer architecture. It is often used for system programming and hardware manipulation. Assembly code directly corresponds to machine instructions, which makes it highly efficient but also complex and difficult to write compared to higher-level languages.

Basics

  • Registers - Working with CPU registers. MOV AX, 1 ; Load 1 into AX register
  • Memory Access - Direct memory addressing. MOV [BX], AX ; Store value of AX in memory address held by BX
  • Arithmetic Operations - Basic arithmetic in Assembly. ADD AX, BX ; AX = AX + BX SUB AX, BX ; AX = AX - BX
  • Immediate Values - Use constants in Assembly. MOV AX, 10 ; Load constant 10 into AX register
  • Conditional Jumps - Jump based on conditions. JE label ; Jump if equal JNE label ; Jump if not equal
  • Labels - Define labels for jumps. start: ; Label definition MOV AX, 5 JMP start ; Jump to start label
  • Looping - Basic loop structure in Assembly. MOV CX, 5 ; Loop 5 times loop_start: DEC CX JNZ loop_start ; Jump if CX is not zero
  • Stack Operations - Push and pop operations. PUSH AX ; Push AX onto the stack POP AX ; Pop value from the stack into AX
  • System Calls - Making system calls. MOV AH, 4Ch ; Exit function INT 21h ; Call interrupt 21h to exit program
  • Comments - Writing comments in Assembly code. ; This is a comment in Assembly

Intermediate

  • Functions - Creating functions in Assembly. call_function: ; Function code here ret ; Return from function
  • Data Segment - Define variables in the data segment. .data msg db 'Hello, Assembly!', 0
  • Stack Frame - Manage stack frames for function calls. push bp mov bp, sp ; Access arguments and local variables pop bp
  • Interrupts - Using hardware interrupts for system calls. MOV AH, 9 MOV DX, offset msg INT 21h ; Print string using interrupt
  • Multi-byte Operations - Handling multi-byte numbers. MOV AX, 0x1000 MOV BX, 0x2000 ADD AX, BX ; AX = AX + BX (16-bit addition)
  • String Operations - Manipulate strings in Assembly. MOV SI, offset msg MOV DI, offset result MOV CX, 5 REP MOVSB ; Copy string using REP prefix
  • System Memory Allocation - Allocate and manage memory. MOV AH, 48h ; Allocate memory MOV BX, 10 ; Number of bytes INT 21h ; Call interrupt 21h
  • Bitwise Operations - Bit-level manipulation. AND AX, BX ; AX = AX AND BX OR AX, BX ; AX = AX OR BX
  • Pointer Arithmetic - Work with pointers in Assembly. LEA SI, [BX + 4] ; Load address of BX + 4 into SI
  • Flag Register - Modify processor flags. CLC ; Clear carry flag STC ; Set carry flag

Advanced

  • Calling Conventions - Understanding function calls and parameter passing. ; Push arguments in reverse order for stack-based calling conventions
  • Position-independent Code (PIC) - Write code independent of its address. MOV EAX, [EBX]
  • Exception Handling - Handle exceptions in low-level Assembly. MOV AX, 0x00 ; Clear error flags
  • Macros - Define reusable code snippets. %macro PRINT 1 MOV DX, %1 MOV AH, 9 INT 21h %endmacro
  • Direct Memory Access (DMA) - Access memory directly. MOV AX, 0x1000 MOV ES, AX
  • Memory Mapped I/O - Access hardware through memory addresses. MOV AX, 0x3000 ; Memory-mapped I/O address
  • Accessing BIOS Services - Use BIOS functions via interrupts. MOV AH, 0x0E MOV AL, 'A' INT 10h ; BIOS function to display character
  • Advanced Loops - Create advanced loop structures. MOV CX, 100 .loop: DEC CX JNZ .loop
  • Optimization Techniques - Optimize Assembly code. ; Use registers as much as possible
  • Compiler Directives - Use specific directives for optimization. %include 'macros.asm'

Master

  • Kernel-Level Programming - Write Assembly code for kernel operations. ; Write system calls and handle interrupts directly
  • Inline Assembly in C - Combine Assembly with C for performance. asm("MOV AX, 1");
  • Memory Management at the System Level - Handle memory allocation at the system level. MOV EAX, 0x0800 ; Allocate memory block of size 0x0800 bytes MOV EBX, 0x0400 ; Memory address for use
  • Accessing Hardware Directly - Interface with hardware directly using Assembly. MOV DX, 0x60 ; Hardware port for keyboard input IN AL, DX ; Read from the hardware port into AL register
  • Real-Time Systems Programming - Develop real-time applications in Assembly. MOV CX, 10 ; Real-time task counter LOOP_REALTIME: DEC CX JNZ LOOP_REALTIME
  • Implementing Virtual Memory - Write code to implement virtual memory in Assembly. MOV CR3, PAGE_TABLE_ADDR ; Load the page table address into CR3
  • Advanced Interrupt Handling - Manage custom interrupts in Assembly. MOV AH, 0x25 ; Set interrupt vector MOV AL, 0xFF ; Interrupt number MOV DX, handler ; Address of the interrupt handler INT 21h ; Set up interrupt
  • Disassembling Code - Use Assembly to disassemble machine code. MOV AX, 0x5A00 ; Instruction to disassemble
  • Creating Bootloaders - Write Assembly code for creating bootloaders. MOV AX, 0x7C00 ; Load bootloader address MOV DS, AX ; Set data segment to bootloader
  • Advanced SIMD Operations - Implement SIMD (Single Instruction, Multiple Data) operations in Assembly. MOVAPS XMM0, [memory] ; Load data into XMM0 register
  • Optimizing Assembly for Multi-core Systems - Writing parallel code in Assembly for multi-core processors. MOV EAX, 0x01 ; Processor core initialization

Compilers and Tools for Assembly

  • MASM (Microsoft Macro Assembler) - A widely used assembler for Windows applications.
  • NASM (Netwide Assembler) - A popular assembler for Linux and Windows.
  • GAS (GNU Assembler) - Part of the GNU toolchain, commonly used on Unix-like systems.
  • FASM (Flat Assembler) - An open-source assembler that produces very efficient code.
  • TASM (Turbo Assembler) - An assembler used primarily for DOS-based programming.
  • YASM - A highly flexible and easy-to-use assembler for multiple platforms.
  • GDB (GNU Debugger) - A debugger that supports debugging Assembly programs.
  • Emu8086 - An emulator for learning and testing Assembly code for 8086 microprocessors.
  • Bochs - An emulator used to emulate x86 systems, including Assembly programming.
  • RadASM - A powerful IDE for developing Assembly code.
