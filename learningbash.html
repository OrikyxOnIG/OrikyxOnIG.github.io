Bash is a powerful shell used in Linux and Unix-like systems for command-line tasks. It's widely used for automation, system administration, and scripting, allowing users to perform operations such as file manipulation, process control, and network communication. With its scripting capabilities, Bash allows users to write complex programs to automate repetitive tasks.

Basic Bash Commands

pwd ls cd mkdir touch cp mv rm rmdir

File Permissions

chmod chown : chgrp

Text Manipulation

cat less grep "" sed -i 's///g' nl sort wc echo "text" > echo "text" >>

Process Management

ps ps aux kill pkill top

Variables and Scripts

variable_name="value" echo $variable_name nano script.sh chmod +x script.sh ./script.sh ./script.sh arg1 arg2

Loops

for i in {1..5}; do echo "Number $i"; done counter=1; while [ $counter -le 5 ]; do echo "Counter: $counter"; ((counter++)); done counter=1; until [ $counter -gt 5 ]; do echo "Counter: $counter"; ((counter++)); done

Conditional Statements

if [ $x -eq 10 ]; then echo "x is 10"; fi if [ $x -gt 10 ]; then echo "x is greater than 10"; else echo "x is less than or equal to 10"; fi if [ $x -gt 10 ]; then echo "x is greater than 10"; elif [ $x -eq 10 ]; then echo "x is equal to 10"; else echo "x is less than 10"; fi
