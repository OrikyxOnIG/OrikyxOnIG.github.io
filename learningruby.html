Learning Ruby

Home

Ruby is a dynamic, open-source programming language with a focus on simplicity and productivity. It was designed with an emphasis on being easy to read and write, using a clean and readable syntax. Ruby is used for web development (primarily with Ruby on Rails), scripting, and various other types of applications.

Basics

  • Variables - Declare and use variables. name = "John Doe" puts name
  • Data Types - Strings, Integers, Floats, Arrays, Hashes. number = 42 string = "Hello, Ruby!" array = [1, 2, 3] hash = { "name" => "John", "age" => 30 }
  • String Interpolation - Embed variables in strings. name = "Ruby" puts "Hello, #{name}!"
  • Conditionals - If, else, and elsif statements. age = 18 if age >= 18 puts "You are an adult." else puts "You are a minor." end
  • Loops - Iterate over data. for i in 1..5 puts i end
  • Arrays - Working with arrays. arr = [1, 2, 3, 4, 5] arr.each { |num| puts num }

Intermediate

  • Methods - Define and call methods. def greet(name) puts "Hello, #{name}!" end greet("Alice")
  • Blocks - Use blocks with methods. 3.times { puts "Hello!" }
  • Hashes - Use key-value pairs. person = { "name" => "John", "age" => 30 } puts person["name"]
  • Regular Expressions - Pattern matching in strings. text = "The Ruby programming language is great!" if text =~ /Ruby/ puts "Match found" end
  • Ranges - Working with ranges. (1..5).each { |i| puts i }
  • File Handling - Reading and writing to files. File.open("file.txt", "w") { |file| file.puts "Hello, Ruby!" } content = File.read("file.txt") puts content

Advanced

  • Object-Oriented Programming (OOP) - Define classes and objects. class Person def initialize(name, age) @name = name @age = age end def greet puts "Hello, #{@name}!" end end person = Person.new("John", 30) person.greet
  • Inheritance - Creating child classes. class Employee < Person def initialize(name, age, job_title) super(name, age) @job_title = job_title end def greet super puts "I am a #{@job_title}." end end employee = Employee.new("Alice", 25, "Developer") employee.greet
  • Modules - Reuse code across multiple classes. module Speak def speak puts "Hello!" end end class Person include Speak end person = Person.new person.speak
  • Exception Handling - Handle errors. begin raise "An error occurred" rescue => e puts "Error: #{e.message}" end
  • Enumerable Methods - Working with collections. arr = [1, 2, 3, 4] puts arr.map { |n| n * 2 } puts arr.select { |n| n.even? }
  • Procs and Lambdas - Create callable objects. greeting = Proc.new { |name| puts "Hello, #{name}!" } greeting.call("Alice") square = ->(x) { x * x } puts square.call(4)

Expert

  • Metaprogramming - Write code that writes code. class MyClass define_method(:say_hello) { puts "Hello!" } end obj = MyClass.new obj.say_hello
  • Monkey Patching - Add or modify methods in existing classes. class String def reverse_words self.split.reverse.join(" ") end end puts "Hello World".reverse_words

  • Fiber - Manage lightweight concurrency. fiber = Fiber.new do puts "Fiber started" Fiber.yield puts "Fiber resumed" end fiber.resume fiber.resume
  • Custom Iterators - Create your own iterators. class MyIterator def initialize @index = 0 end def each yield 1 yield 2 yield 3 end end it = MyIterator.new it.each { |item| puts item }

Compilers and Tools for Ruby

  • RubyMine - IDE for Ruby development.
  • IRB - Interactive Ruby Shell for testing and exploring Ruby code.
  • RVM - Ruby Version Manager for managing multiple Ruby versions.
  • Rails - Full-stack web application framework written in Ruby.
  • Bundler - A tool for managing Ruby dependencies.
  • Sinatra - Minimalist web framework for Ruby.
  • RSpec - A testing framework for Ruby.
  • JRuby - Ruby on the Java Virtual Machine (JVM).
  • Rubinius - A Ruby implementation focused on performance.
  • Repl.it - An online Ruby IDE and compiler.
